Easy Route Platform

Manage your loan company with Easy Route Platform

EasyRoute is a loan route manager for customers in the Dominican Republic, it helps loan routes owners to have full control over ther bussines.

This is actually my most robust project, I developed EasyRoute platform under a multitenancy arquitecture, one database per tenant; I took advantage of the powerfull spring boot framework and JWT to create a mechanism that allows me to identify each user when doing login and conect it to its respective database, this was challenging for me but i did it.

I also developed an other api using spring boot to set a payment system for tenants wich execute a method every day to confirm that each tenant has paid its subscription otherwise the system blocks the tenant and not letting them sign in again until they pay their subscription.

How it works?

I have two apis one for the back-end operations and the other to check the payment status of each tenant, on the other hand i have two android apps, one for admins and the other for employees, admins have access to all the permissions within the api and employees only have a few of them.

Each app is conected to the main api that has all the bussines logic, for this approach I used Retrofit 2, when a user does login the api creates a unique jwt that identifies the tenant wich allows the api perform the operations in its respective database.

This is the database schema for each tenant

Deployment

I deployed the apis in an AWS Ec2 instance where they are actually running, this platform is already in production and I have some customers using it.
